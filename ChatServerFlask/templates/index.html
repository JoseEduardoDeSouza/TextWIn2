Objetivos

O objetivo deste nosso projeto de pesquisa é a criação de um novo método de comunicação móvel, que permite a troca de dados (arquivos, mensagens de texto) mesmo em falta ou inoperância de rede convencional (Internet, rede de dados celular, e também sem Wi-Fi local).


Justificativa e Vantagens

Independência e/ou ampliação dos serviços de comunicações (Internet, redes de dados celular). Possível aplicação em redes 6G.

Redução dos custos com serviços de comunicações (Internet, redes de dados celular). Possível aplicação em redes 6G.

Possibilidade de comunicação em desastres naturais (ex. terremoto). Internet e rede de dados celular fica inoperante
